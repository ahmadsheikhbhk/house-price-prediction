# House Price Prediction

Predicting house prices using machine learning regression models in Python.

## Dataset
California Housing Dataset (built into scikit-learn) — 20,640 samples

## Models Used
- Linear Regression
- Ridge Regression
- Random Forest Regressor

## Results
| Model | R² Score | RMSE |
|-------|----------|------|
| Linear Regression | 0.83 | 0.52 |
| Ridge Regression | 0.83 | 0.52 |
| Random Forest | 0.81 | 0.53 |

## How to Run
1. Install libraries: pip install pandas numpy matplotlib seaborn scikit-learn
2. Run: python house_price_prediction.py
